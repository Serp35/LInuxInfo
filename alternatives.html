Why Should I use Free Software?

The free software has these advantages:

No tracking

It shows you the code so anyone can make sure they are not tracking you.

Free to use

People can download and distribute the software freely.
-

No bugs

it is constantly tested and modified by the community, for free, just to help.

No ads

Nobody makes profit from it, this means therer won't be advertisement.

Faster

As it does not track you, and it gets updated to improve, not to make profit, it is faster.

Open Source is not Free Software!

Free Software:

- You can read the source code
- The source code can be distributed and shared
- You can modify the source code for any purpose
- You can distribute the modified code to anyone

Open Source:

- You can read the source code
- The source code can't be distributed and shared
- You can't modify the source code for any purpose
- You can't distribute the modified code to anyone
